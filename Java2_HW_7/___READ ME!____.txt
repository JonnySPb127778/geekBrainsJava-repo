Т.к. была необходимость ввода настроек до запуска сети, из инициализации запуск соединения с сервером пришлось 
убрать, а соединение выполнять по кнопке "Connect".

    Для быстрого понимания ниже приведена краткое описание для работы с клиентским приложением.

1. Элементы настройки сети: "Server" и "Port" можно ввести в ручную, если не вводить то по нажатию кнопки "Connect"
в силу вступят настройкм по умолчанию.

2. Элементы настройки "Nickname"
Служит для ввода имени пользователя в чате. Если не вводить имя до запуска соединения, то по кнопке "Connect" 
приложение сгенерирует его само по шаблону "UserXXX", где "XXX" число от 100 до 999. В процессе работы Nickname 
можно сменить набрав новое имя в поле ввода и нажав "Enter" или кнопку "Choose". Имя смениться, сервер уведомит 
об этом всех участников чата.

3. Элемент  "Message"
Поле ввода текста сообщения. По кнопке "Send" сообщение данного поля будет напрвлено адресату согласно значению 
в поле "Addressee".

4. Элемент "Addressee" (адресат)
По умолчанию установлено "ALL", т.е. "ВСЕМ". При необходимости напрвить приватное сообщение, в это поле вводится имя 
адресата. Для быстрого возврата данного поля ввода в исходное состояние предусмотрена клавиша "ALL".

5. Элементы "In Chat"
В малом поле показывает число участников чата, в большом -- список. Список начинается с имени пользоватеря с 
припиской "(You)", т.е. это Вы. Далее указаны имена остальных участников чата. При появлении, убывании и смене имён 
участников чата, оба поля автоматически обновляются.

6. Поле вывода сообщений чата  "Messages"
6.1 Все сообщения участников чата выводятся в следующем формате: Nickname отпрвителя, разделитель "@", Nickname 
получателя, разделитель "> ", далее текст сообщения.
    Например:
Bob@ALL> Hello! How are you Girls!!!

6.2. При выводе сообщений программа клиента автоматически подменяет Nickname пользователя, как в поле отпрвителя, 
так и в поле получателя, на слово "You".
6.3 Сообщения от сервера выводятся без форматирования.

7. Кнопка "Disconnect". по нажатию выдается команда серверу на выход, закрыватся сокет клиента, внутренние входящие 
и исходящие стримы.
Примечания:
1. Действия по данной кнопке не имеют обраного действия по кнопке "Connect".
2. Действия по данной кнопке не приводят к появлению сообщения у участников чата о том что пользователь покинул чат.
   Данное сообщение появляется, если грубо прервать выполнение программы клиента.
